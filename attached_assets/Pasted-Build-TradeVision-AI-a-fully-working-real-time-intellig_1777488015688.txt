## BOT ENGINE (paper trading, real prices)

Every 30s each active bot evaluates using live cached prices:

**Momentum Bot:** when signal confidence >80% fires BUY. Auto exit when price hits target or SL. Records trade in bot_trades with real NSE price.

**Grid Bot:** user sets range (e.g. 24000–25000) and grids (e.g. 10). Bot buys 1 unit at each grid level below current price, sells at grid above. Uses real NIFTY price from cache.

**DCA Bot:** every N minutes, buys ₹X worth of chosen stock at current live price. Tracks average cost correctly.

**Options Expiry Bot:** Thursday only. At 09:20 reads real ATM strike from options cache. Simulates selling straddle. At 15:15 closes. Records P&L using real LTP from options chain.

All bot P&L, win rate, trade count update in real time → push via Socket.io `bot_update` event.
